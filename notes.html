Notes App

This is a single page application built with react , node and mongo. It is used to save user's notes . Each user will be able to view his/her own notes and cannot view others.
Features :

  • Each note can be given a priority
  • Every priority will have a unique color
  • The user can add new notes , edit exisiting ones and delete notes
  • Each user can have any number of notes and each node can have max 50chars
